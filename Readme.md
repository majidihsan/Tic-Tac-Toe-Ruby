##Tic Tac Toe game in Ruby

```ruby
board_a = [
  ['x', 'o', 'x'],
  ['x', nil, 'o'],
  ['x', 'o', nil]
]
```

Sample output of the `print_grid` method for `board_a` above.

```no-highlight
 x | o | x
-----------
 x |   | o
-----------
 x | o |  
```

```ruby
board_b = [

  ['x', 'o', nil],
  ['x', 'o', nil]
]
```

```no-highlight
   | o | x
```
